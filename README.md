# Alameen Pywinauto Automation

Automation scripts for selected workflows in **Al-Ameen ERP** using **Pywinauto**.

This project focuses on automating specific desktop operations inside Al-Ameen to reduce repetitive work, speed up data entry, and improve overall efficiency.

---

## 🚀 Overview

This repository contains Python scripts that automate parts of the Al-Ameen ERP desktop application.  
The automation is built using **Pywinauto**, enabling reliable interaction with Windows UI elements such as:

- Menus  
- Dialogs  
- Form fields  
- Buttons  
- Data-entry workflows  

The goal is to shorten execution time and eliminate repetitive manual steps.
